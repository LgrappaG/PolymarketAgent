# Code of Conduct

This project is committed to providing a welcoming and inclusive environment for all contributors.

## Our Pledge

We are dedicated to providing a harassment-free experience for everyone, regardless of:
- Age
- Body size
- Disability
- Ethnicity
- Gender identity and expression
- Level of experience
- Nationality
- Personal appearance
- Race
- Religion
- Sexual identity and orientation
- Socioeconomic status

## Our Standards

### Expected Behavior

- Use welcoming and inclusive language
- Be respectful of differing opinions
- Accept constructive criticism gracefully
- Focus on what's best for the community
- Show empathy toward other community members
- Help each other learn

### Unacceptable Behavior

- Harassment, discrimination, or exclusion
- Trolling, insulting/derogatory comments
- Unwelcome sexual attention or advances
- Publishing private information without consent
- Other conduct that could reasonably be considered inappropriate

## Scope

This Code of Conduct applies to all project spaces, including:
- GitHub issues and pull requests
- Comments and discussions
- Code reviews
- Any official project communication

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported by opening an issue or contacting the maintainers privately.

All complaints will be reviewed and investigated promptly. We are committed to maintaining confidentiality where requested.

## Consequences

Violations may result in:
- Warning
- Temporary or permanent exclusion from project participation
- Reporting to GitHub or other authorities if appropriate
